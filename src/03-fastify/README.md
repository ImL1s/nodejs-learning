# 學習路線 03：Fastify 高性能框架

## 📚 學習目標

- 理解 Fastify 的優勢和特點
- 掌握 Schema 驗證
- 學習插件系統
- 理解性能優化技巧
- 比較 Fastify vs Express

## 📖 為什麼選擇 Fastify？

- ⚡ **高性能** - 比 Express 快約 2-3 倍
- 🔒 **Schema 驗證** - 內建 JSON Schema 驗證
- 🧩 **插件系統** - 更好的代碼組織
- 📝 **TypeScript 友好** - 優秀的類型支持

## 🚀 快速開始

```bash
npm run dev -- src/03-fastify/examples/01-fastify-hello.ts
```

## 💡 重要概念

### Schema 驗證
Fastify 使用 JSON Schema 進行自動驗證和序列化。

### 插件系統
通過插件組織代碼，更好的模組化。
